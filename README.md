
# Introduction
该LaTex模板按照[《中国矿业大学研究生学位论文撰写规定及模板（2021年版）》](https://gs.cumt.edu.cn/info/1049/3149.htm)的要求编写（以下简称《撰写规定》）。目前支持硕士、博士毕业论文撰写（暂不支持外文学院硕士毕业论文）。

By C.Zhang & S.Li
